# POWERBI-DASHBOARD
# Title

## BlinkIT Grocery Sales Analysis Dashboard

An interactive Power BI dashboard developed to analyze BlinkIT grocery sales data, identify product performance trends, evaluate outlet efficiency, and generate actionable business insights through data visualization.

---

# Project Description

The BlinkIT Grocery Sales Dashboard is a comprehensive business intelligence solution built using Power BI. The dashboard provides a 360-degree view of sales performance across products, outlets, and operational metrics.

The project transforms raw grocery sales data into meaningful visual insights that help stakeholders understand customer preferences, outlet performance, product category contribution, and factors affecting sales.

The dashboard consists of four interactive pages:

* Executive Summary
* Product Analysis
* Outlet Analysis
* Operational Insights

Users can dynamically filter data using slicers and drill down into specific business segments for deeper analysis.

# Business Problem Statement

BlinkIT operates through multiple outlet formats and offers a wide range of grocery products. The management team requires a centralized reporting system to monitor sales performance, evaluate outlet efficiency, and understand customer buying behavior.

The challenge is to convert large volumes of transactional data into actionable insights that support strategic business decisions and operational improvements.

The dashboard addresses this challenge by providing real-time visibility into sales trends, product performance, outlet effectiveness, and operational drivers.
